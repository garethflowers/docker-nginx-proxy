# Nginx Reverse Proxy

Docker based Nginx instance configured for use as a Reverse Proxy.

## How to use this image

### Start a Nginx instance

To start a container use the following:
```sh
docker run \
	--name my-nginx-proxy \
	garethflowers/nginx-proxy
```
